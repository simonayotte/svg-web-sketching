Bienvenue

Ceci est la page de présentation du guide utilisateur, celui-ci a pour but de vous guider dans l'utilisation de Polydessin.
Le guide utilisateur est accessible a partir de la page d'acceuil (bouton "Afficher le guide d'utilisation") ainsi que de la page de dessin (bouton ) et peut-être fermé en cliquant en dehors de la fenêtre du guide.

Sujets

La liste des sujets abordés se situe sur le côté gauche de la fenêtre du guide, certains d'entre-eux sont rassemblés par catégorie.
Pour afficher le contenu d'un sujet, cliquez sur le nom du sujet dans la partie gauche de la fenêtre du guide. Cliquer sur une catégorie (nom en gras) permet de révéler les sujets appartenant à cette catégorie.
Il est possible de naviguer entre les sujets avec les boutons "Précédent" et "Suivant" situés au bas de la page su sujet actuel.

Créer un dessin

Il est possible de créer un nouveau dessin en appuyant sur le bouton "Créer un dessin" sur la page principale.
Pour créer un dessin, il suffit de remplir le formulaire qui permet de modifier les dimensions et la couleur d'arrière plan du dessin.
Compléter le formulaire emmène sur la page de dessin. Les outils de dessin se trouvent dans la partie gauche de la vue de dessin.
Il est possible d'ouvrir le modal de création de dessin avec le raccourci "CTRL + O".

Galerie

La fenêtre modale de Galerie peut-être accédée en appuyant sur le bouton "Ouvrir la galerie" sur la page d'acceuil ou l'icone () dans la page de dessin.
Celle-ci contient la liste de tous les fichiers sauvegardés sur le serveur.
Chaque élément de la liste comporte un nom, des étiquettes (s’il y en a), et une représentation du dessin en format réduit.
Il est possible de filtrer les dessins par étiquettes (option ajouter un filtre).
Il est possible de charger un dessin en pesant sur le bouton de chargement ().
Il est possible de supprimer un dessin en pesant sur le bouton de corbeille ().

Crayon

L'outils crayon permet de dessiner des traits avec des épaisseurs différentes.
Le crayon est accessible en appuyant sur la touche "C", l'épaisseur du trait est modifiable en utilisant la glissière.
Pour dessinner un trait, un clic gauche maintenu suffit. Le crayon utilise la couleur principale.

Pinceau

L'outils pinceau permet de dessiner des traits avec des textures différentes.
Le pinceau est accessible en appuyant sur la touche "W", les textures et l'épaisseur du trait sont accessibles dans la barre de l'outils pinceau.
Pour dessinner un trait, un clic gauche maintenu suffit. Le pinceau utilise la couleur principale.

Ligne

L'outils ligne permet de tracer une ligne d'un ou plusieurs segments.
Pour ce faire, séléctionnez l'option ligne dans la barre d'outils () ou pesez sur la touche "L".
Une fois l'outils séléctionné, un clic définis le début de la ligne et chaque clic suivant forme un segment connecté au clic précédant.
Un double-clic permet de terminer la ligne.

Alignement

Il est possible de créer des segments aligés avec l'axe des X et des angles multiple de 45 degrés en gardant la touche "Shift" enfoncée

Annuler

Il est possible d'annuler le tracage complet avec la touche "Escape" ou de supprimer le dernier point avec la touche "Backspace".

Rectangle

L'outils rectangle permet de dessiner des formes rectangulaires.
Pour ce faire, séléctionnez l'option rectangle dans la barre d'outils () ou pesez sur la touche "1".
Une fois l'outils séléctionné, un glisser-déposer permet de dessiner un rectangle.

Épaisseur

L'épaisseur du contour du rectangle peut-être modifiée avec la glissière "Épaisseur".

Styles

3 styles de rectangles sont disponibles:
- Contour: seul les contours du rectangle sont dessinés. ()
- Plein avec contour: l'intérieur et les contours du rectangle sont dessinés. ()
- Plein: seul l'intérieur du rectangle est dessiné. ()

Couleur

La couleur principale définie la couleur de l'intérieur du rectangle.
La couleur secondaire définie la couleur du contour du rectangle.

Carré

Il est possible de dessiner un carré plutôt qu'un rectangle en maintenant enfoncée la touche "Shift".

Polygone

L'outils polygone permet de dessiner des polygones avec un glisser-déposer.
Pour ce faire, séléctionnez l'option polygone dans la barre d'outils () ou pesez sur la touche "3".

Paramètres

L'épaisseur du contour du polygone peut-être modifiée avec la glissière "Épaisseur".
3 styles de polygones sont disponibles:
- Contour: seul les contours du polygone sont dessinés. ()
- Plein avec contour: l'intérieur et les contours du polygone sont dessinés. ()
- Plein: seul l'intérieur du polygone est dessiné. ()
Le nombre de côtés du polygone peut-être modifiée avec la glissière "Côtés".

Ellipse

L'outils ellipse permet de dessiner des ellipses avec un glisser-déposer.
Pour ce faire, séléctionnez l'option ellipse dans la barre d'outils () ou pesez sur la touche "2".

Paramètres

L'épaisseur de l'ellipse peut-être modifiée avec la glissière "Épaisseur".
3 styles d'ellipse sont disponibles:
- Contour: seul les contours de l'ellipse sont dessinés. ()
- Plein avec contour: l'intérieur et les contours de l'ellipse sont dessinés. ()
- Plein: seul l'intérieur de l'ellipse est dessiné. ()
Il est possible d'en faire un cercle en pesant sur la touche "Shift".

Efface

L'outils efface permet de supprimer des formes du canvas.
Pour ce faire, séléctionnez l'option efface dans la barre d'outils () ou pesez sur la touche "E".
Une fois l'outils séléctionné, un simple clic ou un glisser-déposer permet d'effacer les formes. (les formes a effacer sont entourées de rouge)

Taille

Il est possible de définir la taille de l'efface avec la glissière "Épaisseur".

Aérosol

L'outils efface permet de dessinner un jet de peinture vaporisé.
Pour ce faire, séléctionnez l'option aérosol dans la barre d'outils () ou pesez sur la touche "A".
Une fois l'outils séléctionné, maintenir le bouton gauche de la souris enfoncé permet de vaporiser.

Paramètres

Il est possible de définir le diamètre du jet de l'aérosol avec la glissière "Épaisseur".

Il est possible de définir le nombres d'émissions par seconde du jet de l'aérosol avec la glissière "Taux d'émission".

Pipette

L'outils pipette permet de choisir une couleur déja présente sur le dessin.
Pour ce faire, séléctionnez l'option pipette dans la barre d'outils () ou pesez sur la touche "I".
Une fois l'outils séléctionné, appuyer sur le clic gauche permet d'assigner la couleur principale et clic droit la couleur secondaire.

Applicateur de couleur

L'outils applicateur de couleur permet de changer la couleur d'un objet déja dessiné.
Pour ce faire, séléctionnez l'option applicateur de couleur dans la barre d'outils () ou pesez sur la touche "R".
Une fois l'outils séléctionné, appuyer sur le clic gauche sur un objet permet de colorier l'intérieur avec la couleur principale et l'extérieur avec la couleur secondaire actuelle.

Sélection

L'outils selection permet de selectionner les objets déja dessinés et de les déplacer.
Pour ce faire, séléctionnez l'option selection dans la barre d'outils () ou pesez sur la touche "S".
Une fois l'outils séléctionné, appuyer sur le clic gauche sur un objet permet de le sélectionner.
Il est possible de sélectionner plusieurs objets avec un glisser-déposer, tous les objets touchant le réctangle de sélection ainsi dessiné seront selectionnés.
Appuyer sur clic droit plutôt que clic gauche permet d'inverser la selection.
Il est possible de sélectionner tous les objets a l'écran en pesant sur "Ctrl + A".

Déplacement

Il est possible de déplacer une sélection en faisant un glisser déposer en partant de l'intérieur de la boîte englobante des objets sélectionnés.
Il est possible de déplacer une sélection en pesant sur les flèches directionelles du clavier.

Sauvegarder

Il est possible de sauvegarder un dessin.
Pour ce faire, séléctionnez l'option sauvegarder dans la barre d'outils () ou pesez sur les touches "Ctrl + S".
Il est ensuite possible de rentrer un nom ainsi que plusieurs étiquettes, ces dernières sont optionelles.

Exporter

Il est possible d'exporter localement un dessin.
Pour ce faire, séléctionnez l'option exporter dans la barre d'outils () ou pesez sur les touches "Ctrl + E".
Il est possible d'exporter l'image dans les formats JPG, PNG et SVG.
Il est possible de rentrer le nom du fichier ainsi que d'appliquer un filtre au dessin avant de l'exporter.

Annuler-refaire

Il est possible d'annuler ou refaire les dernières actions effectuées sur un dessin.
Pour ce faire, séléctionnez l'option annuler dans la barre d'outils () ou pesez sur les touches "Ctrl + Z".
Pour ce faire, séléctionnez l'option refaire dans la barre d'outils () ou pesez sur les touches "Ctrl + Shift + Z".

Grille

Il est possible d'afficher une grille sur la surface de dessin.
Pour ce faire, séléctionnez l'option grille dans la barre d'outils () ou pesez sur la touche "G".
il est possible de définir la taille de la grille avec le champ "Taille Grille".
il est possible de définir l'opacité de la grille avec la glissière "Opacité Grille".
